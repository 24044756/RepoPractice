RosePanel (Java)

RosePanel is a Java Swing project that visually displays a beautiful rose using custom graphics. It uses Java Graphics and Graphics2D libraries to draw the rose pattern programmatically. The project focuses on GUI rendering, coordinate plotting, and color styling to create an artistic visual effect.

  • Draws a symmetrical rose using Java graphics
  • Demonstrates use of loops, math functions (like sine and cosine), and colors
  • Simple, visually appealing output
  • Great introduction to 2D graphics in Java

Tools Used: NetBeans IDE 8.2
